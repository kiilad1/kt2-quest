# Лабиринт — текстовый квест

Текстовая игра-квест на C++. 
Вы играете за персонажа, который оказался в подземном лабиринте. Цель — выбраться на поверхность, решив различные головоломки.

## Описание

В лабиринте всего **4 локации**, отображенные в терминале с помощью ASCII-символов:

| Локация | Состояние | Задача |
|---------|-----------|-------------|
| Вход в лабиринт | открыта (начало) | — |
| Зал с колоннами | закрыта | математическая задача |
| Библиотека | закрыта | загадка |
| Выход на поверхность | закрыта | Перестановка букв |

Из входа можно попасть в зал или же библиотеку. Все промежуточные локации ведут к выходу. Попадание в открытый выход — победа.

## Сущности

- **Персонаж** — хранит текущую локацию; умеет взаимодействовать с закрытыми (головоломки) и открытыми (перемещение) локациями.
- **Локация** — ID, ASCII-форма, связи с другими локациями, состояние (открыта/закрыта), головоломка (если закрыта).
- **Головоломка** — три разных типа: математическая, загадка, перестановка букв в правильном порядке.

## Требования

- Компилятор C++17 или новее (g++, clang++ или MSVC)
- Windows / Linux / macOS

## Сборка и запуск

### Visual Studio (MSVC) — Windows

```cmd
cl /EHsc /std:c++17 /utf-8 main.cpp
chcp 65001
main.exe
```

Флаг `/utf-8` обязателен и нужен для корректного отображения русского текста в консоли.

### g++ (MinGW / Linux / macOS)

```bash
g++ -std=c++17 -o labyrinth main.cpp
./labyrinth
```

На Windows:

```bash
g++ -std=c++17 -o labyrinth.exe main.cpp
labyrinth.exe
```

## Как играть

После запуска доступно меню:

1. **Решить головоломку** — выбрать соседнюю локацию и открыть ее, ответив на вопрос.
2. **Перейти** — перейти в доступную соседнюю локацию.
3. **Показать локацию** — показать текущую локацию.
0. **Выход** — закончить игру.
